# Salary & Expense Tracker

The main goal of this project is to create a functional dashboard where users can add their monthly salary, track expenses, and view the remaining balance in real time.

## Features
- Add monthly salary
- Add expense name and amount
- Automatic balance calculation
- Input validation for empty and negative values
- Delete expense using trash icon
- Data saved using LocalStorage
- Pie chart visualization using Chart.js
- Low balance warning message
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage
- Chart.js

## How It Works
1. User enters monthly salary
2. User adds expenses one by one
3. Total expenses and remaining balance update automatically
4. Expense data remains saved even after page refresh
5. Pie chart shows Total Expenses vs Remaining Balance
